# packme
A WIP CLI tool for creating/managing minecraft modpacks. Similar to packwiz but with some nifty features specialized to my needs/wants!

## Build Instructions
Simply install [zig](https://ziglang.org/) then run
``` 
zig fetch
zig build
```
